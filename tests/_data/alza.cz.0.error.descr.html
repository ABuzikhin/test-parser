Speed class
Rychlostní třída, známá jako Speed Class, je standardizovaná klasifikace, která definuje minimální zápisovou rychlost paměťové karty. Existují různé třídy, například 2, 4, 6 a 10, kde číslo udává minimální rychlost zápisu v MB/s. Vyšší rychlostní třída znamená schopnost zapisovat data rychleji, což je klíčové pro záznam videa ve vyšším rozlišení nebo pro náročnější aplikace.

Memory card interface
UHS (Ultra High Speed) Class is a standard that refers to higher speed classes of memory cards. UHS-I, UHS-II and UHS-III are different variants that provide different minimum write speeds, ranging from 104 MB/s to 624 MB/s. These cards are ideal for professional video and photography applications where fast and reliable performance is required. UHS classes are also accompanied by U1 and U3 classes, which indicate specific minimum speeds of 10 MB/s and 30 MB/s respectively.
UHS-I
UHS-I (Ultra High Speed Class 1) is a speed class technology standard that can reach speeds of up to 104 MB/s. It is suitable for demanding tasks such as Full HD or 3D video recording. For those who work with large amounts of data or need fast transfer speeds, UHS-I is the ideal choice.
, U3
U3 is a higher standard that provides a minimum write speed of 30 MB/s. When it comes to 4K video recording or high-speed burst capture, U3 is the ideal choice. For professional users who need fast and stable performance, U3 may be essential.
 Done!
Video Class
Video Class is a standard that specifies the minimum sequential write speed for optimal video recording. Different classes such as V6, V10, V30, V60 and V90 specify a minimum write speed in MB/s, which is crucial to ensure smooth video recording at higher resolutions and quality.
V30
The V30 standard with a minimum write speed of 30 MB/s is the ideal choice for professional 4K video recording. It provides ample space for high bitrates and allows you to record video at higher bit rates. It is suitable for professional content creators and film producers.
 Done!
Application speed
Application Performance (AP) refers to a category of memory cards that are optimized for fast access and random read/write data during application startup and operation. These cards are ideal for devices where application runtime performance is critical, and meet specified minimum parameters for different performance levels (such as A1 or A2). This class of card ensures smooth launch and operation of applications directly from the memory card.
A2
A2 (Application Performance Class 2) offers higher speeds with a minimum of 4000/2000 IOPS for random read/write. These cards are designed for more demanding applications where performance is critical, such as games and high performance applications.
